# NotPogMC - a Minecraft-themed Discord game bot
NotPogMC is a game bot themed after Minecraft on Discord. Created in June 2026, it was initially made in February 2022 but then shut down in August 2023.

Invite NotPogMC to your server [[here|https://discord.com/oauth2/authorize?client_id=1516681554858868766&permissions=6193448067909696&integration_type=0&scope=bot+applications.commands]]!

After inviting NotPogMC to your server, type `m!help` in Discord to start.
